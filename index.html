Este site foi construído usando o fastpages, uma plataforma de fácil conversão de Jupyter Notebooks para postagens de Blog.

Através disso, ao publicar uma linha de código específico e permitir o compartilhamento, é possível divulgar toda a informação em modo aberto e com suporte ao Colab, conforme mostra a imagem abaixo:

Isso automatiza o processo de criação de repositórios via ações em meu GitHub, deixando de lado as etapas, conversas e scrypts caracteristicos. Uma lista completa de recursos desde sistema pode ser acessada em GitHub para aqueles que desejarem usar o mesmo método.
